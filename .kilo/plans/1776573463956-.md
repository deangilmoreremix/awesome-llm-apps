# Plan: Verify Code Presence in All LLM Apps

## Goal
Check every listed app in the "Awesome LLM Apps" repository to ensure each has actual code files (e.g., Python scripts, config files) and is not just an empty directory. This will confirm usability by verifying the presence of runnable code, dependencies (requirements.txt), and documentation (README.md).

## Background
The repository is structured with apps in subdirectories under categories like `ai_agent_tutorials/`, `rag_tutorials/`, etc. The main README lists approximately 50+ apps, but some may have sub-variants, totaling over 100. Each app should ideally contain:
- Source code files (.py, .js, etc.)
- requirements.txt or pyproject.toml
- README.md with setup instructions
- Optional: config files, data, or Docker setups

## Methodology
1. **Extract App List**: Parse the main README.md to build a complete list of apps across all categories (AI Agents, RAG, MCP, Memory, Chat-with-X, Finetuning, Advanced Tools).
2. **Categorize and Group**: Organize apps by category for efficient checking. Use explore agents to parallelize checks since there are many apps.
3. **Check Each App Directory**:
   - Verify the subdirectory exists.
   - Scan for code files (use glob for patterns like `*.py`, `*.js`).
   - Check for key files: README.md, requirements.txt.
   - Note any missing elements or empty directories.
4. **Use Explore Agents**: Launch multiple "explore" subagents (one per category or group of 10-15 apps) to search thoroughly. Each agent will:
   - List subdirectories in the category folder.
   - Read key files (e.g., README, requirements) to confirm content.
   - Report findings with file paths and summaries.
5. **Aggregate Results**: Compile a report showing complete apps, partial apps (e.g., code but no README), and empty ones. Suggest fixes for missing elements.

## Steps

3. **Review Agent Outputs**: Analyze results for patterns (e.g., all Python-based, common missing files).
4. **Generate Final Report**: Summarize in the plan file or separate doc, highlighting any apps needing fixes.
